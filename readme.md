# OptimizePress 2 Affiliate Royale compatilibity plugin

##Description

Description: Fixes incompatibility issues caused by both plugins using FLOT JS library for charting. Deregisters OP Stats dashboard widget.

##Installation

1. Click Download ZIP button on the right
2. Install plugin archive through WordPress Add plugin screen
3. Activate the plugin
